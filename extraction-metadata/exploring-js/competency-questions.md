# Competency Questions for Exploring JavaScript

Source: "Exploring JavaScript" by Dr. Axel Rauschmayer

## Definitional (What is X?)

1. What is a variable in JavaScript and how do `let`, `const`, and `var` differ?
2. What are primitive values vs. objects in JavaScript?
3. What is the `typeof` operator and what does it return for each type?
4. What is `undefined` and how does it differ from `null`?
5. What is a template literal?
6. What is a Symbol and when would you use one?
7. What is an arrow function?
8. What is a Promise?
9. What is an async function?
10. What is a module in JavaScript (ECMAScript modules)?
11. What is a class in JavaScript?
12. What is the prototype chain?
13. What is destructuring?
14. What is an iterable and what is the iteration protocol?
15. What is a generator function?

## Relational (How does X relate to Y?)

1. How do `==` (loose equality) and `===` (strict equality) relate to type coercion?
2. How does the prototype chain connect objects to their constructors and `Object.prototype`?
3. How do `Map` and `Set` differ from plain objects and arrays as collections?
4. How do callbacks, Promises, and async/await relate as async patterns?
5. How do named exports and default exports relate in the module system?
6. How does `for-of` relate to the iteration protocol?
7. How do synchronous and asynchronous iteration protocols differ?
8. How do `WeakMap`/`WeakSet` relate to `Map`/`Set` regarding garbage collection?
9. How does subclassing (`extends`) interact with the prototype chain?
10. How do rest parameters and spread syntax relate to each other?

## Procedural (How do I do X?)

1. How do I declare and use variables with proper scoping?
2. How do I convert between JavaScript types (string, number, boolean)?
3. How do I handle errors using try/catch/finally?
4. How do I create and use a class with inheritance?
5. How do I import and export modules?
6. How do I create and consume a Promise?
7. How do I use async/await to write asynchronous code?
8. How do I use destructuring to extract values from objects and arrays?
9. How do I use array methods like `.map()`, `.filter()`, and `.reduce()`?
10. How do I write and use regular expressions in JavaScript?
11. How do I work with JSON (stringify and parse)?
12. How do I use `for-of` loops with iterables?

## Prerequisite (What must I know before X?)

1. What must I understand before learning about closures?
2. What concepts are prerequisite to understanding Promises?
3. What must I know before learning about classes and inheritance?
4. What must I understand before working with async iteration?
5. What concepts underpin the module system?
6. What must I know before using typed arrays?
7. What is needed to understand generator functions?

## Diagnostic (What distinguishes X from Y?)

1. What distinguishes `let` from `const` from `var`?
2. What distinguishes primitive values from objects?
3. What distinguishes an arrow function from a traditional function?
4. What distinguishes `null` from `undefined`?
5. What distinguishes `Map` from a plain object used as a dictionary?
6. What distinguishes synchronous from asynchronous code execution?
7. What distinguishes `for-of` from `for-in`?
8. What distinguishes a named export from a default export?
9. What distinguishes `Promise.all()` from `Promise.race()` from `Promise.any()`?
10. What distinguishes shallow copying from deep copying of objects?
