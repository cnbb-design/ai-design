# Domain Taxonomy for Exploring JavaScript

## Categories

- **language-background**: History, evolution, design philosophy, ecosystem context (Ch. 3-7)
- **syntax-fundamentals**: Basic syntax, statements, expressions, comments, strict mode (Ch. 8-9)
- **tooling**: Consoles, assertions, testing, exercises, development tools (Ch. 10-12)
- **variables-scope**: Variable declarations, scoping, hoisting, closures (Ch. 13)
- **types-values**: Type system, primitives vs objects, type checking, coercion (Ch. 14-15)
- **primitive-types**: undefined/null, booleans, numbers, bigints, strings, symbols (Ch. 16-24)
- **control-flow**: Conditionals, loops, iteration statements, labeled statements (Ch. 25)
- **error-handling**: Exceptions, Error classes, try/catch/finally, error chaining (Ch. 26)
- **functions**: Function declarations/expressions, arrow functions, parameters, `this`, closures (Ch. 27-28)
- **modules**: ECMAScript modules, imports/exports, packages, dynamic imports (Ch. 29)
- **objects**: Object literals, properties, methods, prototype chain, copying, descriptors (Ch. 30)
- **classes**: Class declarations, constructors, fields, methods, inheritance, mixins (Ch. 31)
- **iteration**: Iteration protocol, iterables, iterators, for-of, generator functions (Ch. 32-33)
- **collections**: Arrays, typed arrays, Maps, WeakMaps, Sets, WeakSets (Ch. 34-39)
- **destructuring**: Object/array destructuring, defaults, nested patterns (Ch. 40)
- **async-programming**: Event loop, callbacks, Promises, async/await, async iteration (Ch. 41-45)
- **standard-library**: RegExp, Date, JSON (Ch. 46-48)
- **web-development**: Browser APIs, frameworks, tools, next steps (Ch. 49)

## Tiers

- **foundational**: Core concepts with no prerequisites from this source
  - Ch. 3-9: Language background and syntax basics
  - Ch. 13-14: Variables and values fundamentals
  - Ch. 16: undefined and null
  - Ch. 17: Booleans basics

- **intermediate**: Requires foundational concepts
  - Ch. 10-12: Tooling (requires syntax)
  - Ch. 15: Operators (requires types/values)
  - Ch. 18-19: Numbers and Math (requires operators)
  - Ch. 22: Strings (requires primitives)
  - Ch. 25-26: Control flow and exceptions (requires variables, types, operators)
  - Ch. 27: Callable values (requires variables, scope, types)
  - Ch. 29: Modules (requires functions, objects)
  - Ch. 30: Objects (requires values, functions)
  - Ch. 31: Classes (requires objects, functions, prototype chain)
  - Ch. 32: Synchronous iteration (requires objects, functions)
  - Ch. 34: Arrays (requires iteration, types)
  - Ch. 36: Maps (requires objects, iteration)
  - Ch. 38: Sets (requires objects, iteration)
  - Ch. 40: Destructuring (requires objects, arrays)
  - Ch. 41-43: Async foundations and Promises (requires functions, error handling)
  - Ch. 46: Regular expressions (requires strings, objects)
  - Ch. 48: JSON (requires objects, strings)

- **advanced**: Requires intermediate concepts
  - Ch. 20: Bigints (requires numbers)
  - Ch. 21: Unicode (requires strings)
  - Ch. 23: Template literals (requires strings, functions)
  - Ch. 24: Symbols (requires objects, property keys)
  - Ch. 28: Dynamic code evaluation (requires functions, scope)
  - Ch. 33: Generators (requires iteration, functions)
  - Ch. 35: Typed Arrays (requires arrays, numbers)
  - Ch. 37: WeakMaps (requires Maps, garbage collection)
  - Ch. 39: WeakSets (requires Sets, garbage collection)
  - Ch. 44: Async functions (requires Promises)
  - Ch. 45: Async iteration (requires sync iteration, async functions)
  - Ch. 47: Dates (requires objects)
