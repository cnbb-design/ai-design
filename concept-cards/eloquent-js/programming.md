# Quick Definition

Programming is the act of constructing a program -- a set of precise instructions telling a computer what to do.

# Core Definition

As defined in "Eloquent JavaScript" (Ch 0, line 48 of 00-introduction.md): "*Programming* is the act of constructing a *program* -- a set of precise instructions telling a computer what to do. Because computers are dumb, pedantic beasts, programming is fundamentally tedious and frustrating." The skill of programming is described as "the skill of building programs that don't confuse the programmer" (line 143).

# Prerequisites

This is a foundational concept with no prerequisites within this source.

# Key Properties

1. Programming involves constructing **precise instructions** for a computer.
2. It requires rigor -- thinking in terms that machines can process.
3. It is fundamentally about **managing complexity** -- the best programs do something interesting while remaining easy to understand.
4. Programming can be rewarding: it automates tasks, extends what computers can do, and involves puzzle-solving and abstract thinking.

# Construction / Recognition

## To Construct/Create:
1. Identify a task for the computer to perform.
2. Express precise instructions in a programming language.
3. Manage the resulting complexity so the program remains understandable.

## To Identify/Recognize:
1. Any activity that involves writing instructions for a computer to execute.

# Context & Application

Programming is the foundational activity underlying all software development. Understanding it as the construction of precise instructions establishes the mindset needed for learning any programming language, including JavaScript.

# Common Errors

- **Error**: Assuming programming is solely about writing code.
  **Correction**: Programming encompasses the full activity of constructing precise instructions, including understanding the problem, managing complexity, and designing solutions.

# Common Confusions

- **Confusion**: Programming and coding are entirely distinct activities.
  **Clarification**: While "coding" often refers narrowly to writing code, in the context of this source, programming encompasses the broader activity of instruction construction, which includes coding.
