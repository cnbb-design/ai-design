# Quick Definition

Typography is the visual component of the written word — everything about how text looks when it is rendered visually, whether on paper, screen, or billboard.

# Core Definition

Butterick defines typography simply: "Typography is the visual component of the written word." He distinguishes text (a sequence of words that stays the same regardless of rendering) from typography (the visual form that text takes when displayed). "All visually displayed text involves typography — whether it's on paper, a computer screen, or a billboard."

He positions typography as primarily utilitarian rather than artistic: "Is typography an art? ... at their core, both photography and typography have a utilitarian function. The aesthetic component is separate. Being an effective typographer is more about good skills than good taste. And you can learn the skills."

# Key Properties

1. **Visual, not textual**: Typography concerns appearance, not content — the same text can have different typography.
2. **Encompasses all visual properties**: Not just font choice — "Typography encompasses all the considerations about the visual appearance of type. Fonts are one part of that."
3. **Utilitarian at core**: Skills-based, not taste-based — effective typography can be learned.
4. **Universal**: Applies to all visually displayed text, from highway signs to wedding invitations.

# Construction / Recognition

## To Identify/Recognise:
1. Any visual rendering of text involves typographic choices.
2. When two renderings of the same text look different, the difference is typography.

# Context & Application

- **Typical contexts**: Every document, webpage, sign, or screen that displays text.
- **Common applications**: Document design, web design, signage, brand identity, UI design.

# Common Errors

- **Error**: Equating typography with font choice alone.
  **Correction**: Typography encompasses all visual properties of text — size, spacing, colour, weight, arrangement — not just the font.

# Common Confusions

- **Confusion**: Typography is an art that requires innate talent.
  **Clarification**: Butterick argues typography is primarily a skill, not an art: "Being an effective typographer is more about good skills than good taste."
