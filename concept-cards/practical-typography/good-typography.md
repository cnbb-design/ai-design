# Quick Definition

Good typography reinforces the meaning of the text — it is measured by utilitarian effectiveness, not by abstract aesthetic merit.

# Core Definition

Butterick's thesis: "Good typography *reinforces the meaning of the text*." Three subsidiary propositions follow:

1. "Good typography is measured by how well it reinforces the meaning of the text, not by some abstract scale of merit."
2. "For a given text, many typographic solutions would work equally well. Typography is not an algebra problem with one correct answer."
3. "Your ability to produce good typography depends on how well you understand the goals of your text, not on taste or visual training."

He concludes: "Typography that is aesthetically pleasant, but that doesn't reinforce the meaning of the text, is a failure. Typography that reinforces the meaning of the text, even if aesthetically unpleasant, is a success."

# Prerequisites

- **Typography** — Understanding what typography is before evaluating its quality.

# Key Properties

1. **Utilitarian benchmark**: Measured by effectiveness, not aesthetics.
2. **Context-dependent**: "Typographic choices that work for one text won't necessarily work for another."
3. **Multiple valid solutions**: No single correct answer for any given text.
4. **Goal-driven**: Depends on understanding the text's purpose, not on visual taste.
5. **Ugly can succeed**: "Does that mean that effective typography can be ugly? Sure."

# Construction / Recognition

## To Identify/Recognise:
1. Does the typography support the text's primary purpose?
2. Can the reader quickly and comfortably extract the intended meaning?
3. Are the most important elements emphasised appropriately?

# Context & Application

- **Typical contexts**: Evaluating any typographic work — documents, websites, signage, presentations.
- **Common applications**: Quality assessment, design critique, self-evaluation of typographic choices.

# Examples

**Example 1** (what-is-good-typography): Highway signs — "The highway-signage font stays legible under all these conditions. It's good typography because it supports the sign's message. The script font may be prettier, but in this context, it's bad typography."

**Example 2** (what-is-good-typography): Speed limit sign — three versions of the same sign with the same font, but only the one that makes "75" the dominant element is good typography because it delivers the message.

# Common Errors

- **Error**: Evaluating typography by how "nice" or "clean" it looks in isolation.
  **Correction**: Evaluate by how well it delivers the text's meaning to the intended audience in the intended context.

# Common Confusions

- **Confusion**: The best typography is "invisible" (the Crystal Goblet thesis).
  **Clarification**: Butterick explicitly dissents: good typography may be highly visible and even aggressive if that's what the text requires.
