# Quick Definition

JavaScript was created in May 1995 in 10 days by Brendan Eich at Netscape, originally as a glue language for Java applets in web browsers.

# Core Definition

"JavaScript was created in May 1995 in 10 days, by Brendan Eich. Eich worked at Netscape and implemented JavaScript for their web browser, Netscape Navigator." (Ch. 5, &sect;5.1). JavaScript was intended as a glue language for Java parts in web pages and had to look like Java, ruling out existing languages like Perl, Python, and TCL. Its name changed from Mocha to LiveScript to JavaScript.

# Prerequisites

Foundational concept with no prerequisites.

# Key Properties

1. Created in 10 days in May 1995
2. Creator: Brendan Eich at Netscape
3. Original purpose: glue language for Java applets in Netscape Navigator
4. Name evolution: Mocha (code name) -> LiveScript (Sept 1995) -> JavaScript (Dec 1995)
5. Had to resemble Java syntactically, ruling out other scripting languages

# Construction / Recognition

Historical context -- not a constructed concept but essential background for understanding JavaScript's design decisions.

# Context & Application

The rapid creation explains many of JavaScript's quirks. The requirement to look like Java explains the C-family syntax despite unconventional semantics.

# Examples

- Netscape Navigator 2.0 beta 3 (December 1995): JavaScript

# Common Errors

- **Error**: Assuming JavaScript is related to Java beyond syntax.
  **Correction**: JavaScript was designed to *look like* Java for marketing purposes, but is fundamentally a different language.

# Common Confusions

- **Confusion**: Thinking "JavaScript" and "Java" are related technologies.
  **Clarification**: The name was a marketing decision; JavaScript has more in common with Scheme and Self than with Java.
