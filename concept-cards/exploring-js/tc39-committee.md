# Quick Definition

TC39 (Ecma Technical Committee 39) is the committee of companies -- including Adobe, Apple, Google, Microsoft, and Mozilla -- that evolves JavaScript through bimonthly meetings and open collaboration.

# Core Definition

"TC39 (Ecma Technical Committee 39) is the committee that evolves JavaScript. Its members are, strictly speaking, companies: Adobe, Apple, Facebook, Google, Microsoft, Mozilla, Opera, Twitter, and others. That is, companies that are usually competitors are working together on JavaScript." (Ch. 5, &sect;5.4). TC39 meets every two months, publishes minutes publicly on GitHub, and collaborates with the broader JavaScript community.

# Key Properties

1. Members are companies, not individuals
2. Competing companies collaborate (Adobe, Apple, Google, Microsoft, Mozilla, etc.)
3. Bimonthly meetings with public minutes on GitHub
4. Collaboration with the JavaScript community outside meetings
5. Open process encouraging public feedback

# Construction / Recognition

TC39 is referenced when discussing language proposals, feature stages, and the standardization process.

# Context & Application

Understanding TC39 explains how JavaScript evolves: no single company controls it, features go through a rigorous proposal process, and the community can provide input.

- **Error**: Thinking one company (e.g., Google or Mozilla) controls JavaScript.
