# Quick Definition

Web development with JavaScript encompasses browser APIs (DOM), HTML/CSS, frontend frameworks, server-side runtimes (Node.js, Deno, Bun), JavaScript tooling, and progressive web apps, forming a vast but learnable ecosystem.

# Core Definition

"Exploring JavaScript" Ch. 49 identifies key areas: "Browser APIs such as the Document Object Model (DOM)... JavaScript-adjacent technologies such as HTML and CSS... Frontend frameworks... JavaScript runtimes... JavaScript tooling... Progressive web apps (PWAs)." The advice: "Focus on the web technologies that you work with most often and learn them well."

# Prerequisites

Foundational orientation concept with no prerequisites.

# Key Properties

1. Browser APIs (DOM) are the foundation of frontend development
2. Frontend frameworks (React, Vue, Svelte, etc.) simplify UI development
3. Server-side runtimes: Node.js (most popular), Deno, Bun
4. WinterCG standardizes API interoperability across runtimes
5. MDN web docs is recommended for learning
6. "Trust in your ability to learn things on demand!"

- **Error**: Trying to learn everything at once
