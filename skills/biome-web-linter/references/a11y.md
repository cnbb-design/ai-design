# Accessibility (a11y)

Accessibility rules for HTML/JSX — ARIA compliance, semantic elements, keyboard navigation.

**34 recommended** (enabled by default) | **1 optional** (enable individually)

## Recommended Rules

### noAriaHiddenOnFocusable

Enforce that aria-hidden="true" is not set on focusable elements.

**Don't:**
```jsx
<div aria-hidden="true" tabIndex="0" />
```

**Do:**
```jsx
<button aria-hidden="true" tabIndex="-1" />
```

---

### noAriaUnsupportedElements

Enforce that elements that do not support ARIA roles, states, and properties do not have those attributes.

**Don't:**
```jsx
<meta charset="UTF-8" role="meta" />
```

**Do:**
```jsx
<meta charset="UTF-8" />
```

---

### noAutofocus

Enforce that autoFocus prop is not used on elements.

**Don't:**
```jsx
<input autoFocus />
```

**Do:**
```jsx
<input />
```

---

### noDistractingElements

Enforces that no distracting elements are used.

**Don't:**
```jsx
<marquee />
```

**Do:**
```jsx
<div />
```

---

### noHeaderScope

The scope prop should be used only on <th> elements.

**Don't:**
```jsx
<div scope={scope} />
```

**Do:**
```jsx
<th scope={scope}></th>
```

---

### noInteractiveElementToNoninteractiveRole

Enforce that non-interactive ARIA roles are not assigned to interactive HTML elements.

**Don't:**
```jsx
<input role="img" />;
```

**Do:**
```jsx
<input role="button" />;
```

---

### noLabelWithoutControl

Enforce that a label element or component has a text label and an associated input.

**Don't:**
```jsx
<label for="js_id" />;
```

**Do:**
```jsx
<label for="js_id" aria-label="A label" />;
<label for="js_id" aria-labelledby="A label" />;
<label htmlFor="js_id" aria-label="A label" />;
<label htmlFor="js_id" aria-labelledby="A label" />;
<label>A label<input /></label>;
<label>A label<textarea /></label>;
<label><img alt="A label" /><input /></label>;
```

---

### noNoninteractiveElementToInteractiveRole

Enforce that interactive ARIA roles are not assigned to non-interactive HTML elements.

**Don't:**
```jsx
<h1 role="button">Some text</h1>
```

**Do:**
```jsx
<span role="button">Some text</span>
```

---

### noNoninteractiveTabindex

Enforce that tabIndex is not assigned to non-interactive HTML elements.

**Don't:**
```jsx
<div tabIndex="0" />
```

**Do:**
```jsx
<div />
```

---

### noPositiveTabindex

Prevent the usage of positive integers on tabIndex property

**Don't:**
```jsx
<div tabIndex={1}>foo</div>
```

**Do:**
```jsx
<div tabIndex="0" />
```

---

### noRedundantAlt

Enforce img alt prop does not contain the word "image", "picture", or "photo".

**Don't:**
```jsx
<img src="src" alt="photo content" />;
```

**Do:**
```jsx
<>
	<img src="src" alt="alt" />
	<img src="src" alt={photo} />
	<img src="bar" aria-hidden alt="Picture of me taking a photo of an image" />
</>
```

---

### noRedundantRoles

Enforce explicit role property is not the same as implicit/default role property on an element.

**Don't:**
```jsx
<article role='article'></article>
```

**Do:**
```jsx
<article role='presentation'></article>
```

---

### noStaticElementInteractions

Enforce that static, visible elements (such as <div>) that have click handlers use the valid role attribute.

**Don't:**
```jsx
<div onClick={() => {}}></div>;
```

**Do:**
```jsx
<>
    <div role="button" onClick={() => {}}></div>
    <span role="scrollbar" onClick={() => {}}></span>
    <a href="http://example.com" onClick={() => {}}></a>
</>
```

---

### noSvgWithoutTitle

Enforces the usage of the title element for the svg element.

**Don't:**
```jsx
<svg>foo</svg>
```

**Do:**
```jsx
<svg>
    <title>Pass</title>
    <circle />
</svg>
```

---

### useAltText

Enforce that all elements that require alternative text have meaningful information to relay back to the end user.

**Don't:**
```jsx
<img src="image.png" />
```

**Do:**
```jsx
<img src="image.png" alt="image alt" />
```

---

### useAnchorContent

Enforce that anchors have content and that the content is accessible to screen readers.

**Don't:**
```jsx
<a />
```

**Do:**
```jsx
<a>content</a>
```

---

### useAriaActivedescendantWithTabindex

Enforce that tabIndex is assigned to non-interactive HTML elements with aria-activedescendant.

**Don't:**
```jsx
<div aria-activedescendant={someID} />
```

**Do:**
```jsx
<div aria-activedescendant={someID} tabIndex={0} />
```

---

### useAriaPropsForRole

Enforce that elements with ARIA roles must have all required ARIA attributes for that role.

**Don't:**
```jsx
<span role="checkbox"></span>
```

**Do:**
```jsx
<span role="checkbox" aria-checked="true"></span>
```

---

### useAriaPropsSupportedByRole

Enforce that ARIA properties are valid for the roles that are supported by the element.

**Don't:**
```jsx
<a href="#" aria-checked />
```

**Do:**
```jsx
<>
    <a href="#" aria-expanded />
    <img alt="foobar" aria-hidden />
    <div role="heading" aria-level="1" />
</>
```

---

### useButtonType

Enforces the usage of the attribute type for the element button

**Don't:**
```jsx
<button>Do something</button>
```

**Do:**
```jsx
<>
    <button type="button">Do something</button>
    <button type={buttonType}>Do something</button>
</>
```

---

### useFocusableInteractive

Elements with an interactive role and interaction handlers must be focusable.

**Don't:**
```jsx
<div role="button" />
```

**Do:**
```jsx
<div role="button" tabIndex={0} />
```

---

### useGenericFontNames

Disallow a missing generic family keyword within font families.

**Don't:**
```css
a { font-family: Arial; }
```

**Do:**
```css
a { font-family: "Lucida Grande", "Arial", sans-serif; }
```

---

### useHeadingContent

Enforce that heading elements (h1, h2, etc.) have content and that the content is accessible to screen readers. Accessible means that it is not hidden using the aria-hidden prop.

**Don't:**
```jsx
<h1 />
```

**Do:**
```jsx
<h1>heading</h1>
```

---

### useHtmlLang

Enforce that html element has lang attribute.

**Don't:**
```jsx
<html></html>
```

**Do:**
```jsx
<html lang="en"></html>
```

---

### useIframeTitle

Enforces the usage of the attribute title for the element iframe.

**Don't:**
```jsx
<iframe />
```

**Do:**
```jsx
<>
  <iframe title="This is a unique title" />
  <iframe title={uniqueTitle} />
  <iframe {...props} />
</>
```

---

### useKeyWithClickEvents

Enforce onClick is accompanied by at least one of the following: onKeyUp, onKeyDown, onKeyPress.

**Don't:**
```jsx
<div onClick={() => {}} />
```

**Do:**
```jsx
<div onClick={() => {}} onKeyDown={handleKeyDown} />
```

---

### useKeyWithMouseEvents

Enforce onMouseOver / onMouseOut are accompanied by onFocus / onBlur.

**Don't:**
```jsx
<div onMouseOver={() => {}} />
```

**Do:**
```jsx
<>
  <div onMouseOver={() => {}} onFocus={() => {}} />
  <div onMouseOut={() => {}} onBlur={() => {}} />
  <div onMouseOver={() => {}} {...otherProps} />
  <div onMouseOut={() => {}} {...otherProps} />
  <div onMouseOver={() => {}} onFocus={() => {}} {...otherProps} />
  <div onMouseOut={() => {}} onBlur={() => {}} {...otherProps} />
</>
```

---

### useMediaCaption

Enforces that audio and video elements must have a track for captions.

**Don't:**
```jsx
<video />
```

**Do:**
```jsx
<audio>
		<track kind="captions" {...props} />
	</audio>
```

---

### useSemanticElements

It detects the use of role attributes in JSX elements and suggests using semantic elements instead.

**Don't:**
```jsx
<div role="checkbox"></div>
```

**Do:**
```jsx
<>
  <input type="checkbox">label</input>
  <hr/>
  <div role="status"></div>
</>;
```

---

### useValidAnchor

Enforce that all anchors are valid, and they are navigable elements.

**Don't:**
```jsx
<a href={null}>navigate here</a>
```

**Do:**
```jsx
<a href="https://example.com" onClick={something}>navigate here</a>
```

---

### useValidAriaRole

Elements with ARIA roles must use a valid, non-abstract ARIA role.

**Don't:**
```jsx
<div role="datepicker"></div>
```

**Do:**
```jsx
<>
  <div role="button"></div>
  <div role={role}></div>
  <div></div>
</>
```

---

### useValidAriaValues

Enforce that ARIA state and property values are valid.

**Don't:**
```jsx
<span role="checkbox" aria-checked="test">some text</span>
```

**Do:**
```jsx
<>
    <span role="checkbox" aria-checked={checked} >some text</span>
    <span aria-labelledby="fooId barId" >some text</span>
</>
```

---

### useValidAutocomplete

Use valid values for the autocomplete attribute on input elements.

**Don't:**
```jsx
<input type="text" autocomplete="incorrect" />
```

**Do:**
```jsx
<>
  <input type="text" autocomplete="name" />
  <MyInput autocomplete="incorrect" />
</>
```

---

### useValidLang

Ensure that the attribute passed to the lang attribute is a correct ISO language and/or country.

**Don't:**
```jsx
<html lang="lorem" />
```

**Do:**
```jsx
<Html lang="en-babab" />
```

---

## Optional Rules

### noNoninteractiveElementInteractions

Disallow use event handlers on non-interactive elements.

**Don't:**
```jsx
<div onClick={() => {}}>button</div>
```

**Do:**
```jsx
<button onClick={() => { }}>button</button>
```
